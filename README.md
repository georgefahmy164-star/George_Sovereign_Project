# George_Sovereign_Project# - SOVEREIGN CORE v10.0

## Overview
A high-performance Android Forensic & Intelligence system built with **Python**, **PyQt6**, and **JNI Reflection**. Designed for deep data extraction and military-grade encryption.

## Key Features
* **Deep Extraction:** Accesses SMS, Contacts, and Call Logs via Android Content Providers.
* **Military Encryption:** AES-256 GCM authenticated encryption for all captured data.
* **Silent Execution:** Runs background threads for non-blocking UI performance.
* **Local Vault:** Secure SQLite database with WAL mode for high-speed I/O.

## Installation
1. Install dependencies: `pip install -r requirements.txt`
2. Build for Android: `buildozer android debug deploy run`
